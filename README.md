# freeCodeCamp: Legacy Responsive Web Design (V8)

This repository contains the five mandatory projects required to complete the **Legacy Responsive Web Design (V8)** certification from [freeCodeCamp](https://www.freecodecamp.org/). 

These projects demonstrate a solid understanding of semantic HTML5, modern CSS3, accessibility best practices, and responsive design techniques (including CSS Flexbox, CSS Grid, and Media Queries) without the use of any external frameworks or libraries.

## 🏆 Certification Projects

### 1. Survey Form
* **Theme:** Tech Conference Feedback Survey
* **Description:** A responsive, user-friendly form designed to collect user input.
* **Key Concepts:** HTML5 form validation, `<input>` types (text, email, number, radio, checkbox), `<select>` dropdowns, semantic `<label>` pairing, and mobile-friendly styling.

### 2. Tribute Page
* **Theme:** Alan Turing
* **Description:** A clean, accessible tribute page honoring a historical figure, featuring an image, timeline, and external links.
* **Key Concepts:** Responsive images using `max-width`, semantic markup (`<main>`, `<figure>`, `<figcaption>`, `<blockquote>`), and basic typography scaling.

### 3. Technical Documentation Page
* **Theme:** Python Basics
* **Description:** A multi-section documentation layout with a fixed navigation menu that seamlessly transitions to a top-bar layout on mobile devices.
* **Key Concepts:** Internal page routing (bookmark links), CSS Media Queries, `position: fixed`, overflow scrolling, and `<code>` snippet formatting.

### 4. Product Landing Page
* **Theme:** Reactify - UI Component Library
* **Description:** A sleek landing page for a fictional tech product featuring an embedded video, a pricing tier layout, and an email signup form.
* **Key Concepts:** CSS Flexbox for layout alignment, fixed headers, embedded `<iframe>` elements, CSS animations, and mock form submission handling.

### 5. Personal Portfolio Webpage
* **Theme:** Developer Portfolio
* **Description:** A personal portfolio website acting as a central hub to showcase previous work and provide contact links.
* **Key Concepts:** CSS Grid (`grid-template-columns`, `minmax`), viewport height (`vh`) sizing for full-screen sections, smooth scrolling, and interactive hover states.

---

## 🛠️ Technologies & Skills Demonstrated

* **HTML5:** Semantic structuring, document flow, and accessibility features.
* **CSS3:** Custom theming, variables, transitions, and animations.
* **CSS Flexbox:** One-dimensional layout structuring for headers, footers, and feature rows.
* **CSS Grid:** Two-dimensional layout structuring for complex grids (e.g., portfolio project galleries).
* **Responsive Web Design:** Mobile-first design principles using `@media` queries to ensure seamless layouts across mobile, tablet, and desktop viewports.
